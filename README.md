<h4>Keybinds:</h4>
<p>wasd -&gt; move</p>
<p>shift -&gt; sprint</p>
<p>f -&gt; interact&nbsp;</p>
<p>leftclick -&gt; shoots if gun is held</p>
<p>rightclick -&gt; aims if gun is held</p>
<p>t -&gt; check ammo</p>
<p>r -&gt; reload</p>
<p>enter -&gt; progress dialogue</p>
<p>tab -&gt; open terminal</p>
<p>esc -&gt; pause menu (possibly removed in future build)</p>
<h4>Commands:</h4>
<p>[inv] -&gt; shows items in inventory with there index number (the number before the name)</p>
<p>[use &lt;item index&gt;] -&gt; uses the selected item (the item index can be found by using [inv]</p>
<p>[drop &lt;item index&gt;] -&gt; drops the selected item (the item index can be found by using [inv]</p>
<p>[help] -&gt; shows help commands</p>
<p>[keybinds] -&gt; get a list of the controls</p>
<p>[commands] -&gt; get a list of all the commands</p>
<p>[hud] -&gt; get the HUD explained and some general advice</p>
<p>[quit] -&gt; exits game")</p>
<h4>HUD:</h4>
<p>The three bars in the right of your screen represent hunger, warmth, and blood.&nbsp;if either of these bars reach zero you will die.&nbsp;Your hunger and blood levels will affect how quickly you loose warmth.&nbsp; print_line("eat food to stay full, sit by a fire to warm up.")</p>
